Tails is a portable live operating system designed for private, anonymous sessions from a USB drive. Key features include:
- Routes network traffic through Tor by default
- Leaves no trace on the computer after shutdown unless persistence is enabled
- Encrypted Persistent Storage for selected files and settings
- Built-in tools for secure communication and document handling
- Strong fit for temporary, travel, or high-risk browsing workflows
