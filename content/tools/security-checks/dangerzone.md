Dangerzone is a practical safety tool for handling documents you do not fully trust. It converts potentially dangerous PDFs, office documents, and images into safer PDFs by rendering them in a sandbox and rebuilding the result locally.

Key features include:
- Converts PDFs, Office documents, and common image formats into safer PDFs
- Uses a sandboxed workflow with no document network access
- Maintained by Freedom of the Press Foundation
- Free and open source
- Useful for journalists, researchers, activists, and anyone receiving risky attachments
- Best treated as document sanitization, not a guarantee that every file is harmless
