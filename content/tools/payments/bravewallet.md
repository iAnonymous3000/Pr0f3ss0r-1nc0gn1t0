Brave Wallet is a self-custody crypto wallet built directly into the Brave browser. Key features include:
- Native wallet experience without a browser extension
- Multi-chain asset support
- Built-in connection to the Brave ecosystem
- Hardware wallet compatibility
- Convenient access for users already standardized on Brave
