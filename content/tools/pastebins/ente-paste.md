Ente Paste is a lightweight encrypted pastebin for handing off short sensitive text without creating an account. Key features include:

- Browser-side encryption before upload
- One-time links that are deleted after a successful open
- Automatic expiry after 24 hours
- Optional password protection
- Preview crawler protections to reduce accidental link consumption

Use Ente Paste for short secrets, temporary instructions, recovery snippets, or API keys that need to move once and then disappear. It is not durable storage, and the full link still needs to be shared through a channel you trust.
