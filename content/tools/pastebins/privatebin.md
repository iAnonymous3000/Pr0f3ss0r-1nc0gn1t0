PrivateBin is an open-source encrypted pastebin where text is encrypted and decrypted in the browser. Key features include:

- Zero-knowledge server design for pasted content
- Self-hosting support
- Password-protected pastes
- Expiration and burn-after-reading options
- Optional discussions, markdown, syntax highlighting, and file uploads

Use PrivateBin when you want a transparent, self-hostable pastebin instead of relying on a generic public paste site. Choose a trusted, maintained HTTPS instance or host it yourself, because each instance still controls the web code your browser runs.
