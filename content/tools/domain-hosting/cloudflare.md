Cloudflare Registrar offers domain registration alongside Cloudflare's DNS, security, and performance tools. Key features include:

- At-cost domain registration and renewal for supported TLDs
- Redacted WHOIS information by default
- Integrated DNS, CDN, DDoS protection, and web security tools
- Cloudflare Pages support for static sites
- Good fit for users who already rely on Cloudflare for site infrastructure
