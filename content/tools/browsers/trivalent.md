Trivalent is a security-focused Chromium-based browser for desktop Linux built by the secureblue project and inspired by Vanadium. Key features include:
- Desktop-relevant hardening influenced by Vanadium
- Stricter defaults around secondary browser features
- Built-in content filtering support
- Good fit for users who want a hardened Chromium option on Linux
- Best fit within secureblue, where it also benefits from SELinux confinement
