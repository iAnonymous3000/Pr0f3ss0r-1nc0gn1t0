Cloudflare Web Analytics gives site owners aggregate traffic and performance metrics without adding a traditional advertising analytics stack. Key features include:

- Cookie-free analytics
- Page views, referrers, device, and geography metrics
- Performance insights through Cloudflare's web tooling
- Works especially well when Cloudflare is already part of the site stack
- Good fit for privacy-focused personal sites and small publications
