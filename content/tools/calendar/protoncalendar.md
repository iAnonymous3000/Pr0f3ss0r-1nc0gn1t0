Proton Calendar is a privacy-focused calendar app built around encryption. Key features include:
- End-to-end encryption for event details
- Cross-device access on web and mobile
- Calendar sharing within the Proton ecosystem
- Tight integration with Proton Mail
- Cleaner privacy defaults than mainstream calendar services
