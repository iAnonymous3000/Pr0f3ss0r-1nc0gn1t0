SimpleX Chat is an end-to-end encrypted messenger designed to avoid permanent user identifiers. Key features include:

- No phone number, email address, username, or account ID required
- End-to-end encrypted messages, voice calls, and media
- Decentralized relay design with user-controlled connection addresses
- Open source clients and protocol implementation on [GitHub](https://github.com/simplex-chat)
- Available across mobile and desktop platforms

SimpleX is a strong fit when metadata minimization matters as much as message content encryption.
