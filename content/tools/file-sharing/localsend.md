LocalSend is an open-source app for sending files and messages to nearby devices over your local network. Key features include:

- No account, login, or external server required
- Works without an internet connection
- Cross-platform support for Windows, macOS, Linux, Android, and iOS
- Encrypted transfers between devices on the same network
- Useful for quick transfers between your own devices or trusted nearby people

Use LocalSend when the devices are on the same local network and you want a private AirDrop-like workflow across platforms. For remote sharing by link, use Wormhole or OnionShare instead.
