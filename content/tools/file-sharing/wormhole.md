Wormhole is a low-friction web app for sending files with end-to-end encryption and automatically expiring links. Key features include:

- No account required for one-off file transfers
- Browser-side encryption before upload
- Expiring share links with download limits
- Peer-to-peer transfers when both people keep the page open
- Large-file sharing without leaving files in a long-term cloud drive

Use Wormhole when you want something easy enough for normal file sharing. Because the hosted app is not fully open source, treat it as a convenience pick rather than a highest-risk sharing workflow. For stronger anonymity and a fully open-source option, use OnionShare.
